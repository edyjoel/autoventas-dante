# Imágenes del Hero Slider

Coloca las siguientes imágenes en este directorio:

## Imágenes Requeridas

- `hero-1.jpg` - Imagen principal para el slide de inventario (vehículos premium)
- `hero-2.jpg` - Imagen para el slide de financiamiento
- `hero-3.jpg` - Imagen para el slide de calidad/garantía

## Especificaciones Recomendadas

- **Formato**: JPG o WebP
- **Dimensiones**: 1920x1080px o superior (16:9 ratio)
- **Tamaño**: Optimizadas para web (< 300KB cada una)
- **Contenido**: Imágenes de alta calidad de vehículos, showroom, o escenas relacionadas con autoventas

## Notas

- Las imágenes deben ser llamativas y de alta calidad
- Considera usar imágenes con espacio en el centro para el texto
- El degradado radial oscurecerá los bordes automáticamente
- La animación Ken Burns aplicará un zoom sutil
